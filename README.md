# Game : RiverBorn

把抽象的点数比拼，变成具象的战斗生存。

你的底牌（2张）： 两个隐藏的冒险者（比如一个战士、一个带治疗技能的法师），只有你自己知道他们的属性和技能组合。

公共池（5张）： 每张可能是一个职业或者费用的卡牌 逐渐翻开， 翻开的过程中每个玩家可以根据自己的底牌跟场上的牌加注

怪物/Boss池(2-3张)： 也是随着公共牌的翻开而以此翻开 （比如：第1张是哥布林，第2、3张是毒沼泽和宝箱或者精英怪，或者关底Boss）


加注博弈（核心玩法）： 随着怪物池和公共池的翻开，玩家就要根据自己手里的“底牌”+公共池里的三张牌组合成的队伍能否打通当前的怪物池顺便吃下他人的手牌和队伍，来进行加注（押金币/补给）。

如果你觉得你的阵容会被毒沼泽克制，你可以弃牌（撤退），损失之前的投入。

如果你觉得你的阵容无敌，或者你想诈唬（Bluff）吓退别人，你可以All-in。

结算： 坚持到最后的人，让自己的冒险者下地牢。打通了，赢走所有池子里的钱；打不过（翻车了），即便别人被你吓退了，你也只能空手而归甚至受到惩罚。

## 奖池设计
1）主池（玩家下注池）

来自所有玩家的：

ante
call
raise
all-in

这是大家真正想抢的“大钱”。

2）地下城收益池（副本奖励）

来自：

击败怪物
宝箱
Boss 掉落
事件奖励

这个池子可以是金币、补给、遗物、声望等。

结算建议：
情况 A：无人通关
本轮主池流入“地下城”/滚入下轮 Jackpot
所有坚持到底的人都白给
弃牌的人只损失已投入的筹码
营造“这一把大家都死了”的残酷感

这个很有节目效果。

情况 B：只有 1 人通关
他直接拿走主池
再拿地下城收益池
是完美单吃
情况 C：多人通关
地下城收益池：可以按通关者共享，或者按通关表现分配
主池：由“通关者中的最终 PvP 胜者”拿走

这样就能同时体现：

“过本能力”
“吃人能力”

## 底牌，公共池和怪物池的设计

1）底牌（私有）

每人 2 个隐藏冒险者
例如：

战士（嘲讽）
神官（治疗）

只有自己知道。

2）公共池（共 5 张，逐步翻）

公共池不是“扑克牌点数”，而是：

冒险者
装备
技能
消耗品
招募位 / 费用位 / 事件位

你最后要从：

2 张底牌
5 张公共牌

里构筑出一支队伍（比如选 5 个单位/或 3 个单位+2个装备）

这个结构特别像德扑“2 张底牌 + 5 张公共牌组成最佳手牌”。

3）怪物/Boss池（2~3 张，逐步翻）

建议这样对应：

Flop 阶段：翻第一个怪 / 小场景
Turn 阶段：翻第二个环境/精英/宝箱事件
River 阶段：翻最终 Boss

比如：

第1张：哥布林群
第2张：毒沼泽 / 宝箱 / 精英怪
第3张：关底 Boss

这样每次翻牌，玩家都能重新评估自己：

我的阵容打不打得过？
会不会被毒地形克制？
有没有机会吃宝箱收益？
我现在要不要继续压？

这就是德扑味最强的地方。
